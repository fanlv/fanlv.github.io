#微积分基础

## 导数(Derivatives)

y = x^n => dy/dx = nx^n-1

y = sin(x) => dy/dx = cos(x)

y = e^x => dy/dx = e^x

一阶导数为 0 的时候，对应的值不是最大值就是最小值。

## 二阶导数 （Second Derivative）

dy/dx = cos(x) => -sin(x)

y = x^3 - x^2

f'' > 0 定义为凸(convex)对应最小值(局部最小)，f''< 0 的时候定义为凹(concave)对应最大值（局部最大）

## 拐点 （Inflection Point）

拐点就是二阶导数为 0 的点

比较所有驻点（f'=0）处及边界点函数值，得到的最大或者最小值，即函数最值
