# 记录点滴，从现在开始

### ⭐️ 为什么要建这个仓库

以前的我也不喜欢写Bolg，不过随着工作时间越来越久，个人接触的技术栈越来越杂（ASP.NET、WinForm、iOS、Python、Go），我发现半年以上不使用一种技术会很容易就忘记以前开发中一些技术细节，过了一段时间再需要使用的时候又需要去网上查阅相关资料，比如以前做ASP.NET时候做的flash播放器、富文本编辑器，还比如做Socket Server的时候遇到的高并发瓶颈解决方案、Tcp/Udp穿透一些实验结果。还比如iOS开发时对语音通话调研，对FFmpeg在iOS上拉流解码播放的调研，如何解决音视频不同步的问题等等。如果把这些记录下来，可以看做是自己对工作的一个总结，过段时间再看，也可能发现自己以前思考不足的地方，对自己有更好的提高。
